# 프로젝트 개요

## 1. 프로젝트명

WMS (Work Management System)

---

## 2. 프로젝트 목적

기존 Spring MVC + MyBatis 기반 개발 경험에서 벗어나 Spring Boot 기반의 현대적인 웹 애플리케이션 아키텍처를 학습하고 실무형 업무관리시스템을 구축한다.

본 프로젝트는 실제 기업에서 사용 가능한 수준의 SR 관리, 전자결재, 직원관리 기능을 구현하며 설계부터 개발, 테스트, 배포까지 전 과정을 경험하는 것을 목표로 한다.

---

## 3. 개발 목표

* Spring Boot 기반 프로젝트 경험 확보
* Spring Data JPA 활용
* Spring Security 기반 인증/인가 구현
* 실무형 결재 프로세스 구현
* Git/GitHub 기반 형상관리
* Docker 기반 배포 환경 구축
* AWS 서버 배포 경험 확보

---

## 4. 주요 기능

### 대시보드

* 개인 업무 현황 조회
* 결재 대기 현황 조회
* SR 진행 현황 조회

### SR 관리

* SR 요청 등록
* SR 승인/반려
* SR 담당자 지정
* SR 진행 상태 관리

### 전자결재

* 휴가 신청
* 회의실 예약
* 일반 품의
* 결재 승인/반려

### 인사관리

* 직원 등록 신청
* 직원 승인
* 조직도 조회

### 시스템관리

* 메뉴 관리
* 권한 관리
* 공통 코드 관리
* 공지사항 관리

---

## 5. 기술 스택

* Java 21
* Spring Boot 4.0.6
* Spring Security
* Spring Data JPA
* QueryDSL
* MySQL
* Thymeleaf
* Bootstrap
* Docker
* GitHub
* AWS

---

## 6. 개발 방식

요구사항 분석 → 업무 프로세스 정의 → 화면 설계 → ERD 설계 → API 설계 → 구현 → 테스트 → 배포 순으로 진행한다.
