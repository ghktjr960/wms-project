# 메뉴 상세 설명

## 1. 대시보드

로그인 후 최초 진입 화면이다.

### 제공 기능

* 결재 대기 건수 조회
* 진행 중 SR 건수 조회
* 신규 공지사항 조회
* 개인 업무 현황 조회

### 공지사항 영역

대시보드 내 공지사항 영역을 제공한다.

공지사항은 목록 형태로 표시하며 제목 선택 시 상세 내용을 조회할 수 있다.

예시

* 시스템 점검 안내
* 휴가 정책 변경 안내
* 회의실 예약 정책 변경

# 결재 정책

## 기본 정책

* 상신자는 자신의 결재 문서를 승인할 수 없다.
* 모든 승인 및 반려 이력은 저장한다.
* 반려 시 반려 사유 입력은 필수이다.
* 모든 상태 변경 이력은 저장한다.

---

## 대직 정책

결재권자는 대직자를 지정할 수 있다.

대직자는 지정된 기간 동안 원 결재자의 결재를 대신 처리할 수 있다.

### 대직 정보

* 원 결재자
* 대직자
* 시작일
* 종료일
* 사용 여부

### 처리 정책

대직 기간 중 발생한 결재는 대직자가 처리한다.

결재 이력에는 대직 처리 여부를 함께 저장한다.

예시

* 원 결재자 : 김팀장
* 대직자 : 홍차장
* 대직 기간 : 2026-07-01 ~ 2026-07-07

위 기간 동안 발생한 결재는 홍차장이 처리한다.
