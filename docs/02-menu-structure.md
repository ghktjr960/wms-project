# 02. Menu Structure

## 1. 문서 목적

본 문서는 업무관리시스템(WMS)의 메뉴 구조를 정의한다.

메뉴는 업무 목적과 권한(Role)에 따라 구분하며, 메뉴 구조는 화면 설계 및 권한 정책의 기준으로 사용한다.

상세 업무 프로세스는 03.Business Process 문서에서 정의한다.

---

# 2. 전체 메뉴 구조

```text
Dashboard

├ 내 정보

├ SR 관리

│ ├ 요청

│ ├ 진행

│ ├ 현황 조회

│ └ 통계

│

├ 전자결재

│ ├ 결재 작성

│ ├ 결재함

│ └ 결재처리

│

├ 시설관리

│ ├ 시설 예약

│ ├ 시설 예약 확인

│ └ 시설 사용 현황

│

├ 인사관리

│ ├ 직원 조회

│ ├ 조직도 조회

│ ├ 직원 관리

│ └ 직원 등록 요청

│

└ 시스템관리

├ 메뉴 관리

├ 권한 관리

├ 부서 관리

├ 공통코드 관리

├ 시설 관리

├ 공지사항 관리

├ 시스템 설정

├ 시스템 모니터링

└ 시스템 로그 조회
```
