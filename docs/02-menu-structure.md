# 02. Menu Structure

## 1. 메뉴 구성

시스템은 다음 메뉴로 구성한다.

```text
대시보드

SR

전자결재

인사관리

시설관리

시스템관리
```

---

# 2. 대시보드

시스템 로그인 후 최초 진입 화면이다.

## 메뉴

```text
대시보드
```

## 기능

* 공지사항 조회
* 미처리 SR 건수 조회
* 미처리 전자결재 건수 조회
* 시설 예약 현황 조회
* 읽지 않은 알림 건수 조회

---

## 정책

읽지 않은 알림 건수 클릭 시

```text
알림 목록
```

으로 이동한다.

---

# 3. SR

SR은

```text
요청

진행

현황
```

으로 구성한다.

---

## 3.1 SR 요청

### 기능

* SR 등록
* SR 수정
* SR 삭제
* SR 요청
* SR 요청취소
* 첨부파일 등록

---

### 버튼

SAVE

```text
SR 요청

삭제
```

---

REQUEST

```text
SR 요청취소
```

---

## 3.2 SR 진행

개발팀 전용 메뉴이다.

---

### 기능

* SR 승인
* SR 반려
* 담당자 선택
* 개발완료
* 인수테스트 요청
* 인수테스트 승인
* 인수테스트 반려
* 개발팀장 확인

---

### 버튼

DEPT_APPROVED

```text
승인

반려
```

---

DEV_APPROVED

```text
승인

반려
```

승인 시

```text
담당자 선택
```

은 필수 입력이다.

---

IN_PROGRESS

```text
개발완료
```

---

DEV_COMPLETED

```text
인수테스트 요청
```

---

UAT

```text
인수테스트 승인

인수테스트 반려
```

---

DEV_MANAGER_CONFIRMED

```text
확인
```

---

COMPLETED

```text
버튼 없음
```

---

## 3.3 SR 현황

전체 사용자가 조회할 수 있다.

### 기능

* SR 조회
* 진행상태 조회
* 개발자 코멘트 조회
* 엑셀 다운로드

---

### 버튼

```text
조회

초기화

엑셀 다운로드
```

---

# 4. 전자결재

전자결재는

```text
결재함

결재처리
```

로 구성한다.

---

## 4.1 결재함

### 기능

* 결재 등록
* 상신
* 상신취소
* 첨부파일 등록

---

### 상태

```text
SUBMITTED

↓

IN_PROGRESS

↓

COMPLETED

또는

REJECTED

또는

CANCELED
```

---

## 4.2 결재처리

### 기능

* 승인
* 반려

---

### 버튼

```text
승인

반려
```

---

# 5. 인사관리

인사관리는

```text
직원조회

조직도 조회

직원관리

직원등록
```

으로 구성한다.

---

## 5.1 직원조회

### 기능

* 직원 조회
* 직원 상세조회
* 엑셀 다운로드

---

### 정책

직원 상세 조회 시

```text
PERSONAL_INFO_ACCESS_LOG
```

를 저장한다.

---

## 5.2 조직도 조회

### 기능

* 조직도 조회
* 부서 조회
* 직원 조회

---

## 5.3 직원관리

### 기능

* 직원 정보 수정
* 부서 변경
* 비밀번호 초기화
* 로그인 잠금
* 로그인 잠금 해제

---

## 5.4 직원등록

### 기능

* 직원 등록
* 수정
* 삭제
* 등록 요청
* 승인
* 반려

---

### 정책

인사팀은 모든 부서의 신규 직원을 등록할 수 있다.

승인 완료 시

```text
EMPLOYEE
```

테이블에 실제 직원 데이터를 생성한다.

---

# 6. 시설관리

시설은

```text
회의실

차량

교육장
```

으로 구성한다.

---

## 6.1 시설예약

### 기능

* 예약
* 예약취소
* 첨부파일 등록

---

### Action

```text
RESERVE

CANCEL
```

---

### 정책

예약은 물리삭제하지 않는다.

예약취소 시

```text
CANCELED
```

상태로 변경한다.

---

## 6.2 시설사용현황

### 기능

* 예약 현황 조회
* 기간별 조회
* 시설별 조회
* 엑셀 다운로드

---

# 7. 시스템관리

시스템관리 메뉴는 운영 및 환경설정을 담당한다.

---

## 7.1 공지사항 관리

### 기능

* 공지 등록
* 수정
* 삭제
* 팝업 게시
* 팝업 종료

---

## 7.2 권한관리

### 기능

* 권한 등록
* 권한 수정
* 사용자 권한 부여
* 사용자 권한 회수

---

## 7.3 메뉴별 권한관리

### 기능

* 메뉴 조회
* 권한별 메뉴 노출 설정
* 저장

---

### 정책

권한에 따라

```text
Top 메뉴

Left 메뉴
```

노출 여부를 결정한다.

---

## 7.4 코드관리

### 기능

* 그룹코드 등록
* 상세코드 등록
* 명칭 변경
* 사용여부 변경

---

### 정책

그룹코드

```text
대문자 + 언더바

예)

SR_STATUS

EAPP_STATUS
```

---

상세코드

```text
영문 + 숫자
```

---

코드는

* 코드값 변경 불가
* 명칭 변경 가능
* 물리삭제 불가

---

## 7.5 부서관리

### 기능

* 부서 등록
* 부서 수정
* 사용여부 변경

---

## 7.6 시스템 로그

### 기능

* SYSTEM_LOG 조회
* DOWNLOAD_LOG 조회
* PERSONAL_INFO_ACCESS_LOG 조회

---

## 7.7 에러 로그

### 기능

* ERROR_LOG 조회
* 에러 상세 조회
* 에러 발생 화면 조회
