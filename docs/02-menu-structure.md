# 메뉴 구조 설계

## 1. 대시보드

시스템 로그인 후 최초 진입 화면

### 주요 기능

* 공지사항 목록 조회
* 내가 처리해야 할 결재 건수 조회
* 진행 중인 SR 건수 조회
* 시설 예약 현황 조회
* 최근 처리 내역 조회

---

## 2. 내 정보

사용자 개인 정보 관리

### 메뉴

* 개인정보 수정
* 비밀번호 변경
* 대직자 설정

### 대직자 설정

휴가, 출장 등의 사유로 결재 처리가 불가능한 경우 대직자를 지정할 수 있다.

설정 정보

* 대직자
* 시작일
* 종료일
* 대직 사유

---

## 3. SR 관리

시스템 변경 요청 및 진행 현황 관리

### SR 요청

전 직원 사용 가능

기능

* SR 등록
* SR 조회
* SR 수정
* SR 취소

등록 정보

* 제목
* 요청 내용
* 요청 우선순위(긴급 / 일반)
* 첨부파일

---

### SR 진행

개발팀 전용 메뉴

기능

* 담당자 지정
* 담당자 변경
* 담당자 이첩
* 담당자 가져오기
* 상태 변경
* 처리 의견 등록

---

### SR 현황 조회

전 직원 사용 가능

기능

* 진행 상태 조회
* 담당자 조회
* 처리 의견 조회
* 처리 이력 조회

---

### SR 통계

개발팀 및 관리자 사용

기능

* 개발자별 처리 건수
* 개발자별 평균 처리 기간
* 요청 부서별 요청 건수
* 우선순위별 처리 건수
* 월별 처리 현황

---

## 4. 전자결재

전자결재 작성 및 승인

### 결재 작성

기능

* 결재 상신
* 결재 임시저장
* 결재 수정
* 결재 취소

결재 정보

* 결재유형
* 제목
* 내용
* 승인자
* 참조자
* 첨부파일

### 결재유형

결재유형은 공통코드로 관리한다.

초기 제공 유형

* 일반품의
* 휴가신청
* 출장신청
* 외근신청
* 교육신청
* 행사참석
* 기타

신규 결재유형은 시스템관리 > 공통코드 관리에서 추가 가능하다.

---

### 결재함

기능

* 내가 상신한 결재
* 내가 승인해야 할 결재
* 참조된 결재
* 완료된 결재
* 반려된 결재

---

### 결재 처리

승인 권한자만 접근 가능

기능

* 승인
* 반려
* 처리 의견 등록

참조자는 조회만 가능하며 승인, 반려, 의견 작성은 불가능하다.

대직자가 설정된 경우 대직자가 승인 및 반려 처리가 가능하다.

---

## 5. 시설관리

회사 시설 예약 관리

### 시설 예약

기능

* 시설 예약
* 예약 등록
* 예약 수정
* 예약 취소

시설 종류

* 회의실
* 차량
* 교육장

예약 정보

* 시설 종류
* 시설명
* 예약일
* 시작시간
* 종료시간
* 사용목적

동일 시설에 대해 중복 시간 예약은 불가능하다.

---

### 시설 예약 확인

기능

* 내 예약 조회
* 예약 상세 조회
* 예약 취소

---

### 시설 사용 현황

전 직원 조회 가능

기능

* 시설별 사용 현황 조회
* 일별 조회
* 주간 조회
* 월간 조회

---

# 조직 및 권한 체계

## 부서

* 경영관리팀
* 개발팀
* 영업팀
* 기획팀
* 운영팀

부서는 시스템관리 > 부서관리 메뉴에서 추가 및 수정할 수 있다.

---

## 직급

* 사원
* 대리
* 과장
* 차장
* 부장
* 팀장

---

## 시스템 권한

* USER
* ADMIN

---

## 업무 권한

* DEPT_MANAGER
* DEV_MANAGER
* HR_MANAGER
* FACILITY_MANAGER

직급과 권한은 별도로 관리한다.

결재 권한은 직급이 아닌 권한(Role)을 기준으로 처리한다.

직원은 여러 개의 권한을 동시에 보유할 수 있다.
