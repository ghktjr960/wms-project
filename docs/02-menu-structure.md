# 02. Menu Structure

## 1. 메뉴 구성

```text

전자결재

시설관리

시스템관리
```

---

# 2. 대시보드

시스템 로그인 후 최초 진입 화면

## 메뉴

```text
대시보드
```

* 공지사항 조회
* 미처리 SR 건수
* 미처리 결재 건수
* 예약 현황

알림 건수 클릭 시

```text
알림목록
```

으로 이동한다.
